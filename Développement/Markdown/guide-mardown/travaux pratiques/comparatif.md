# Comparatif - Environnements de Développement
## Présentation
Ce document compare trois environnements de développement couramment utilisés en BTS SIO : **VS Code**, **PhpStorm** et **Vim**. Le tableau ci-dessous met en avant leurs caractéristiques principales, afin d'aider à choisir l'outil le plus adapté.
## Tableau comparatif
| Outil    | Type                   | Licence | Difficulté | Points forts                      |
| :------- | :--------------------- | :-----: | :--------: | --------------------------------: |
| VS Code  | Éditeur de code        | Gratuit | Facile     | Extensions, ergonomie, communauté |
| PhpStorm | IDE complet pour PHP   | Payant  | Moyen      | Analyse du code, outils intégrés  |
| Vim      | Éditeur modal terminal | Gratuit | Difficile  | Rapidité, utilisation avancée     |
